# Fitness and Health

This site provides fitness and health tips for an active lifestyle.

## Description

The "Fitness a zdraví" (Fitness and Health) site contains:
- Useful health habits
- Exercise recommendations
- Nutrition tips
- Localization in Czech

## Technologies

- HTML5
- CSS3 (Bootstrap)
- JavaScript

## File Structure

- `index.html` - Main page
- `style.css` - Styles
- `script.js` - Scripts

## Running

Open `index.html` in a browser.
