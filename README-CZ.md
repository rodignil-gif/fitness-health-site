# Fitness a zdraví

Tento web poskytuje rady pro fitness a zdraví pro aktivní životní styl.

## Popis

Web "Fitness a zdraví" obsahuje:
- Užitečné návyky pro zdraví
- Doporučení pro cvičení
- Rady pro výživu
- Lokalizaci v češtině

## Technologie

- HTML5
- CSS3 (Bootstrap)
- JavaScript

## Struktura souborů

- `index.html` - Hlavní stránka
- `style.css` - Styly
- `script.js` - Skripty

## Spuštění

Otevřete `index.html` v prohlížeči.
