# Fitness a zdraví

Этот сайт предоставляет советы по фитнесу и здоровью для активного образа жизни.

## Описание

Сайт "Fitness a zdraví" (Фитнес и здоровье) содержит:
- Полезные привычки для здоровья
- Рекомендации по упражнениям
- Советы по питанию
- Локализация на чешском языке

## Технологии

- HTML5
- CSS3 (Bootstrap)
- JavaScript

## Структура файлов

- `index.html` - Главная страница
- `style.css` - Стили
- `script.js` - Скрипты

## Запуск

Откройте `index.html` в браузере.
